Name Type Description
"Entry No." BigInteger

Specifies the entry number of the label printer setup.

Specifies the label action code that this setup is associated with.

Specifies the action that will print the label.
"Label Code" Code[20]

Specifies the unique code for the label.

Specifies the description of the label.
"Label Printer Code" Code[20]

Specifies the unique code for the label printer.
"User Type" Enum "NICTEK User Type"

Specifies the user type for which this label printer setup is applicable.
"User ID" Code[50]

Specifies the user task group for which this label printer setup is applicable.
"Label Status" Enum "NICTEK Label Status"

Specifies the status of the label. Only labels with a status of "Active" can be printed.
"No. of Labels Source" Enum "NICTEK No. of Labels Source"

Specifies the source for the number of labels to print.
"No. of Labels" Integer

Specifies the number of labels to print.
"Max No. of Labels" Integer

Specifies the maximum number of labels to print when the source is set to Label Table Field Value. This is to prevent excessive label printing.
