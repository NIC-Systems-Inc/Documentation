GetBankSignature

Get a signature from a Bank Account record. The signature is returned only if the amount is within the limit for the signature.

procedure GetBankSignature(BankAccountNo: Code[20], Amount: Decimal, SignatureNo: Integer): Record "Tenant Media"

Parameters

Name Type Description
BankAccountNo Code[20]

The bank account number where the signature exists.
Amount Decimal

The document or transaction amount.
SignatureNo Integer

The number of the signature to return.

Returns

Type Description
Record "Tenant Media"

The signature. The signature is returned only if the amount is within the limit for the signature, otherwise an empty record is returned.
