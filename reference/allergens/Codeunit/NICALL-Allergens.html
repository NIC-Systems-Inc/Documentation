BomHasAllergens

Check if a production bom or production bom line has associated allergens

procedure BomHasAllergens(ProdBomNo: Code[20], VersionCode: Code[20], LineNo: Integer): Boolean

Parameters

Name Type Description
ProdBomNo Code[20]

The production bom number you want to check.
VersionCode Code[20]

The production bom version you want to check. You can leave this blank.
LineNo Integer

The production bom line number you want to check. Leave this as zero to check the entire production bom.

Returns

Type Description
Boolean

True if the production bom or line has associated allergens.

Returns
