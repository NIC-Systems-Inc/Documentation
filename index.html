NIC Systems Developer Documentation

Welcome to the developer documentation for the NIC Systems apps, which use Microsoft Dynamics 365 Business Central. This documentation is designed to help developers and administrators utilize and extend the functionalities of the NIC Systems apps. You will find information on the various components available within our apps.

If you're new to Business Central development, we recommend starting with the Microsoft Dynamics 365 Business Central Developer Documentation to familiarize yourself with the basic concepts and tools.
